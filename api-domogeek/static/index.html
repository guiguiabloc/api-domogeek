Une API pour répondre aux besoins simples de vos scénarios Domotique.

A propos de API Domogeek

Quand on commence à créer des scénarios domotique, il est fréquent d'avoir besoin de réponses à des questions toutes simples du style "Est-ce qu'aujourd'hui est un jour férié ?" "Quelle est la couleur tempo EDF du jour ?" "Quelle est la couleur vigilance Météo du jour ?"...

Je me suis rendu compte que de simples informations basiques dont tout le monde a besoin demande ou un développement personnel de l'utilisateur sur sa box, ou des appels à différents services divers et variés.

L'idée de l'Api Domogeek est donc d'offrir un point d'entrée unique pour répondre à ce type de questions publiques.
